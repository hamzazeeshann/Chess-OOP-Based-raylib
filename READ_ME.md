This is a chess game I made in my second semester using the principles of OOP and the raylib library for the GUI.
All the classes of the pieces and the board are implemented by me. The game includes:
-All the standard chess movements.
-Saving a game and playing it later or start a new game.
-Professional GUI that allows user to click and move pices
-Promotion and castling

The game is running and has no bugs. If you wanna play it and try it out, download this repo as ZIP and run the exe file, all the dependencies are included in the ZIP. You can also see all the code and logic implemented. Thank you.
